GitHub Actions

Sirve para automatizar cosas en GitHub.

IDEA SIMPLE

GitHub Actions = robots automáticos.

Cuando pasa algo:

push
commit
pull request
horario
release

GitHub ejecuta tareas automáticamente.

EJEMPLO REAL

Tú haces:

git push

GitHub detecta:

"hubo un push"

y ejecuta:

.github/workflows/webpack.yml
ESA CARPETA
.github/workflows/

es MUY importante.

Ahí viven los workflows automáticos.

workflow = automatización

Ejemplo:

name: Test Project

on: push

jobs:

  build:

    runs-on: ubuntu-latest

    steps:

      - uses: actions/checkout@v4

      - name: Install dependencies
        run: npm install

      - name: Run project
        run: npm run build
